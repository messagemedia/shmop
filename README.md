This library provides utility classes for manipulating shared memory segments in PHP.

### Installation

#### Composer

```json
{
    "require": {
        "MessageMedia/shmop": "dev-master"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/messagemedia/shmop"
        }
    ]
}
```
