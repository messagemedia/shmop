static MessageMedia\shmop\Packing::getPackFormat (   $id,
  $structure,
  $named = false 
)
static

Create a pack format based on a structure for packing or unpacking.

$structure is expected to be an associative array of name => type pairs.

$id can be any string unique to the $structure parameter. It is used to cache the results, such that calling this function a second time with the $id and $named values, will result in this function returning an array cached from the previous call. This is necessary to prevent a potential (though very small) performance overhead in the metrics classes.

Parameters
$idAn ID to give to this structure so the returned value can be stored for fast retrival in subsequent calls.
$structureThe structure of the data being packed or unpacked.
$namedWhether or not to name each value, ie for use with unpack.
Returns
Format string to be used in pack() and unpack() functions.
